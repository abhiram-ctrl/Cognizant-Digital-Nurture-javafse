# JUnit AAA Pattern

## Objective

Learn how to organize JUnit test cases using the Arrange-Act-Assert (AAA) pattern and use setup and teardown methods.

## Concepts Covered

- @Before
- @After
- @Test
- Arrange
- Act
- Assert
- Test Fixtures

## Execution Order

@Before

↓

Arrange

↓

Act

↓

Assert

↓

@After

## Technology

- Java
- Maven
- JUnit 4.13.2
