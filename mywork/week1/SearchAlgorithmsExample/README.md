# E-commerce Platform Search Function

## Objective

Implement Linear Search and Binary Search algorithms for an e-commerce platform and analyze their performance using Big O notation.

---

## Project Structure

```text
src
├── Product.java
└── SearchTest.java
```

---

## Features

* Product class with:

  * productId
  * productName
  * category

* Linear Search

* Binary Search

* Time Complexity Analysis

---

## Algorithms Implemented

### Linear Search

Checks every element one by one until the product is found.

### Binary Search

Works on sorted data and repeatedly divides the search space in half.
