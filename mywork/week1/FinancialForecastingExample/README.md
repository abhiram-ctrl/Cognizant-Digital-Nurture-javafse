# Financial Forecasting Using Recursion

## Objective

Develop a financial forecasting tool that predicts future values based on historical growth rates using recursion.

---

## Recursion

Recursion is a programming technique where a method calls itself repeatedly until a base condition is reached.

A recursive solution consists of:

1. Base Case
2. Recursive Case

---

## Formula Used

Future Value = Current Value × (1 + Growth Rate)

Repeated recursively for the specified number of years.

---

## Example

Current Value = 10000

Growth Rate = 10%

Years = 3

Result:

13310
