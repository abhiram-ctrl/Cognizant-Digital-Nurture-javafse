# Design Patterns Java

This repository contains implementations of:

1. Singleton Pattern
2. Factory Method Pattern

## Singleton Pattern

Ensures that only one instance of a class exists and provides a global access point to that instance.

Example:

* Logger Utility

## Factory Method Pattern

Provides an interface for creating objects while allowing subclasses to decide which class to instantiate.

Example:

* Document Management System

    * Word Document
    * PDF Document
    * Excel Document

Technologies Used:

* Java
* Object-Oriented Programming
* Design Patterns
