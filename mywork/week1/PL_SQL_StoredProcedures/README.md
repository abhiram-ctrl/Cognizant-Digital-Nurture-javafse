# PL/SQL Stored Procedures

## Objective

Implement PL/SQL stored procedures for common banking operations.

---

## Scenarios

### Scenario 1

Process monthly interest for all savings accounts.

### Scenario 2

Update employee salary by adding a bonus percentage.

### Scenario 3

Transfer money between customer accounts after checking sufficient balance.

---

## Concepts Used

- CREATE PROCEDURE
- Parameters (IN)
- UPDATE
- SELECT INTO
- IF...ELSE
- COMMIT
- DBMS_OUTPUT.PUT_LINE

---

## Technology

- Oracle SQL
- Oracle PL/SQL
