# PL/SQL Control Structures

## Objective

Demonstrate the use of PL/SQL control structures such as:

* IF Statements
* Loops
* Cursors
* UPDATE Statements
* DBMS_OUTPUT

---

## Scenario 1

Apply a 1% loan interest discount to customers above 60 years of age.

---

## Scenario 2

Promote customers with a balance greater than $10,000 to VIP status.

---

## Scenario 3

Generate reminder messages for customers whose loans are due within the next 30 days.

---

## Concepts Used

* DECLARE Block
* CURSOR
* FOR LOOP
* IF Condition
* UPDATE Statement
* COMMIT
* DBMS_OUTPUT.PUT_LINE

---

## Technology

* Oracle PL/SQL
