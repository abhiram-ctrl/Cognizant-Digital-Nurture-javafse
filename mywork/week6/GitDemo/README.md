# Week 6 - Task 1: Git Hands-on Lab

## Objective

This exercise demonstrates the basic Git workflow and configuration.

## Git Commands Practiced

- git --version
- git config --global user.name
- git config --global user.email
- git config --global --list
- git init
- git status
- git add
- git commit
- git push
- git pull

## Tasks Performed

- Verified Git installation.
- Configured Git username and email.
- Initialized a local Git repository.
- Created a sample file (`welcome.txt`).
- Added the file to the staging area.
- Committed the changes to the local repository.
- Pushed the repository to GitHub.

## Folder Structure

```
week6/

```

## Learning Outcomes

- Understand the Git workflow.
- Configure Git for development.
- Track files using Git.
- Commit changes to a local repository.
- Synchronize local and remote repositories using GitHub.
