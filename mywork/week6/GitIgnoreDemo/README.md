# Week 6 - Task 2: Git Ignore

## Objective

Learn how to ignore unwanted files and folders using the `.gitignore` file.

## Git Commands Practiced

- git init
- git status
- git add
- git commit

## Tasks Performed

- Initialized a Git repository.
- Created sample files and folders.
- Created a `.gitignore` file.
- Ignored all `.log` files.
- Ignored the `logs` directory.
- Verified ignored files using `git status`.
- Committed the tracked files.

## Folder Structure

```
GitIgnoreDemo/
├── .gitignore
├── welcome.txt
├── app.log          (ignored)
└── logs/            (ignored)
```

## Learning Outcomes

- Understand the purpose of `.gitignore`.
- Ignore unwanted files and folders.
- Prevent temporary files from being committed.
