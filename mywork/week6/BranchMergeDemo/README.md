# Week 6 - Task 3: Git Branching and Merging

## Objective

Learn how to create branches, make changes independently, merge them into the main branch, and delete merged branches.

## Git Commands Practiced

- git branch
- git checkout
- git switch
- git diff
- git merge
- git log
- git branch -d

## Tasks Performed

- Initialized a Git repository.
- Created a new branch named `GitNewBranch`.
- Switched between branches.
- Modified files in the branch.
- Committed changes.
- Compared differences between branches.
- Merged the branch into the main branch.
- Viewed commit history.
- Deleted the merged branch.

## Folder Structure

```
BranchMergeDemo/
├── README.md
└── file.txt
```

## Learning Outcomes

- Create and manage Git branches.
- Merge feature branches into the main branch.
- Compare branches using Git.
- Visualize commit history.
