# Week 6 - Task 5: Git Cleanup and Remote Synchronization

## Objective

Learn how to synchronize the local repository with the remote repository.

## Git Commands Practiced

- git status
- git branch
- git pull
- git push

## Tasks Performed

- Verified repository status.
- Listed available branches.
- Pulled the latest changes from the remote repository.
- Pushed local commits to GitHub.
- Verified successful synchronization.

## Folder Structure

```
GitCleanupDemo/
└── README.md
```

## Learning Outcomes

- Keep local and remote repositories synchronized.
- Understand the Git pull and push workflow.
- Verify repository status before publishing changes.
