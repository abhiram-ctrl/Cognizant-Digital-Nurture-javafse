# Week 6 - Task 4: Merge Conflict Resolution

## Objective

Learn how merge conflicts occur and how to resolve them manually.

## Git Commands Practiced

- git checkout
- git branch
- git merge
- git status
- git add
- git commit
- git log

## Tasks Performed

- Created a new branch.
- Modified the same file in both branches.
- Generated a merge conflict.
- Resolved the conflict manually.
- Completed the merge.
- Updated `.gitignore`.
- Deleted the merged branch.

## Folder Structure

```
MergeConflictDemo/
├── README.md
├── hello.xml
└── .gitignore
```

## Learning Outcomes

- Understand merge conflicts.
- Resolve conflicts manually.
- Complete conflict-free merges.
- Maintain clean repositories using `.gitignore`.
